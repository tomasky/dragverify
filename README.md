### about 
- This is a typescript build template
- supported typescript 1.x
- build with npm ,tsc and typings
- dependencies: typescript , typings,mocha,chai etc

### usage
- Install
```
$ npm install typescript -g 
$ npm install typings -g 
$ npm install mocha -g 
$ npm link typescript
$ npm install
$ typings install
```
- Compile
```
$ tsc
```

- Unit test  
```
$ npm test
```
